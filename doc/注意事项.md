# 单位换算
## rpm->rad/s
$1rad/s = \pi/30 rpm$

# 数据范围
## 陀螺仪
- yaw $[-\pi,\pi]$
- roll $[-\pi,\pi]$ 
- pitch $[-\frac{\pi}{2},\frac{\pi}{2}]$

## 电机属性
