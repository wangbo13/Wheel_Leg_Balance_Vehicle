## 操作说明
1. 准备模式（遥控器左端拨杆置于`OFF`档，s[1]==1），用于初始化校准小米电机相关参数。
2. 手动提起机器人，使腿部自然下垂接触地面。
3. 正常模式（遥控器左端拨杆置于`CL`档，s[1]==3），接下来理论上会进入平衡状态。
